- Controls
Place Mode: Switches between placing numbers, and making notes on the board.
New Game Button: Option for difficulty, starts a new game.
Clear: Clears user placed numbers.
Solve: Attempts to solve the current board. Won't solve if the board is unsolvable.

- - In Place Mode
Click on a tile, and type the number you wish to be there, then click away, or press enter.
Click and delete a number to clear that tile.
NOTE: In place mode, entering a number on a note will place a number over the note.
NOTE: Clearing a tile will reveal the note under if if there was one.

- - In Note Mode
Click on an empty or yellow tile and type numbers, it will seperate them and keep them there as notes.
(Won't treat them as placed)

- Solving
The board will go green and the timer will stop and say "Solved In:" when the board is solved.

NOTE: Progress is saved when closing the tab! To restart, press new game button.
